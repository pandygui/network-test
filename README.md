## 网速测试软件

原理：一张图片总大小 ÷ 加载一张图片所需的时间

### 编译依赖

`sudo apt install g++ qt5-default libdtkcore-dev libdtkwidget-dev`

### 运行依赖

* qt5widgets

* qt5gui

* qt5core

`sudo apt install libqt5core5a libqt5gui5 libqt5widgets5 libdtkcore libdtkwidget`
